# Fitness Koçu 🏋️

Kişisel 12 haftalık fitness ve beslenme takip uygulaması.

## Özellikler

- 📅 12 haftalık detaylı antrenman programı (3 faz)
- 🏃 Her egzersiz için stickman animasyonları ve form ipuçları
- 🍽 Günlük beslenme planı ve öğün seçenekleri
- 💧 Su takibi ve günlük kontrol listesi
- 📊 Kilo ve vücut ölçüsü takibi
- ⏱ Dinlenme zamanlayıcısı
- 📅 Apple Calendar (ICS) export
- 🖨 Yazdırılabilir haftalık programlar
- 💪 Motivasyon mesajları ve koçluk ipuçları

## Kurulum

```bash
cd fitness-kocu
npm install
npm run dev
```

Tarayıcında [http://localhost:3000](http://localhost:3000) adresini aç.

## Teknolojiler

- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- Framer Motion (animasyonlar)
- Lucide React (ikonlar)
- Recharts (grafikler)
